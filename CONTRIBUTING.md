# Contributing

Contributions are welcome via pull requests on [GitHub](https://github.com/impruthvi/agent-detector-laravel).

## Setup

```bash
git clone https://github.com/impruthvi/agent-detector-laravel.git
cd agent-detector-laravel
composer install
```

## Tests

```bash
composer test
```

## Code style

```bash
composer format
```

## Static analysis

```bash
composer analyse
```

## Guidelines

- Open an issue before submitting a large PR
- Add tests for new behaviour
- Keep PRs focused — one thing per PR
- Run `composer format` and `composer analyse` before pushing
